### **Step 2: 创建多样性级别分类**

#### 方法：
将连续的Gender Diversity (20-80%) 分成5档

```python
df['Diversity_Level'] = pd.cut(
    df['Gender Diversity (%)'],
    bins=[0, 40, 45, 55, 60, 100],
    labels=['低多样性(0-40%)', '中低多样性(40-45%)',
            '均衡(45-55%)', '中高多样性(55-60%)',
            '高多样性(60-100%)']
)
```

#### 为什么这样分？
- **0-40%**: 一方性别占主导（>60%）
- **40-45%**: 略微失衡
- **45-55%**: 相对均衡（"黄金区间"）
- **55-60%**: 略微失衡（另一方向）
- **60-100%**: 另一方性别占主导（>60%）

#### 样本分布：
| 多样性级别 | 样本量 | 占比 |
|-----------|--------|------|
| 低多样性 | 10,034 | 33.4% |
| 中低多样性 | 2,508 | 8.4% |
| 均衡 | 4,988 | 16.6% |
| 中高多样性 | 2,534 | 8.4% |
| 高多样性 | 9,936 | 33.1% |

**观察**：两端（低+高）占比66.5%，中间均衡区仅16.6%

### **Step 4: 计算相关系数 - 量化"多样性红利"**

#### 操作：
```python
# 整体相关性
corr_risk = df['Gender Diversity (%)'].corr(df['Automation Risk (%)'])
corr_salary = df['Gender Diversity (%)'].corr(df['Median Salary (USD)'])
corr_growth = df['Gender Diversity (%)'].corr(df['Openings_Pct_Change'])
```

#### 结果：
| 对比维度 | 相关系数 | 解读 |
|---------|---------|------|
| 多样性 vs 风险 | **r = -0.0031** | 几乎为0（微弱负相关） |
| 多样性 vs 薪资 | **r = -0.0014** | 几乎为0（微弱负相关） |
| 多样性 vs 增长率 | **r = -0.0287** | 微弱负相关 |

#### 核心发现：
> **"多样性红利"在就业风险和薪资维度几乎不存在！**
>
> 相关系数接近0，说明多样性不是风险/薪资的主要驱动因素。

### **Step 6: 行业内部相关性 - 寻找异质性**

#### 操作：
```python
# 各行业内部：多样性 vs 风险 的相关系数
industry_corr_risk = df.groupby('Industry').apply(
    lambda x: x['Gender Diversity (%)'].corr(x['Automation Risk (%)'])

```

#### 结果：
| 行业 | 相关系数 | 解读 |
|------|---------|------|
| **Retail** | **-0.0287** | 多样性↑ → 风险↓ (最强，但仍很弱) |

## 🎯 C3 的核心叙事框架

### 故事线：

```
1. 【引入问题】
   多样性是热点话题，学术界说"多样性有价值"
   → 在AI时代，多样性团队是否更安全？

2. 【数据探索】
   发现1: 所有行业多样性都接近50%（高度均衡）
   发现2: 多样性范围20-80%（没有极端值）

3. 【假设检验】"多样性红利"是否存在？

   测试1: 多样性 vs 风险
   → 相关系数 r = -0.0031 (接近0)

   测试2: 多样性 vs 薪资
   → 相关系数 r = -0.0014 (接近0)

   测试3: 不同多样性级别对比
   → 风险差异<0.5%，薪资差异<0.1%

4. 【反常识发现】
   低多样性岗位增长率最高 (+152%)
   高多样性岗位增长率最低 (+128%)

   → 与"多样性促进创新"假设相反！

5. 【深入分析】行业异质性
   → 所有行业相关系数都极弱 (|r| < 0.03)
   → Retail略显负相关，Manufacturing略显正相关

6. 【解释机制】为什么多样性影响这么小？

   可能原因1: 数据已经高度均衡（49-50%）
   → 缺乏变异，难以检测效应

   可能原因2: 多样性的价值体现在其他维度
   → 比如创新能力、团队氛围、社会价值
   → 而非直接降低自动化风险或提高薪资

   可能原因3: 时间滞后
   → 多样性的长期价值尚未体现在当前数据中

7. 【结论】
   多样性不是就业风险的主要驱动因素
   但这不意味着多样性不重要！

   → 多样性有其他价值（公平、创新、社会责任）
   → 只是在"AI风险规避"这个维度上效应微弱
```

## 📝 写作注意事项

### ✅ 要强调的：
1. **数据发现的客观性**
   - 我们只是呈现数据，不是否定多样性的价值

2. **解释的多元性**
   - 提供多种可能解释（数据均衡、价值维度不同、时间滞后）

3. **局限性的坦诚**
   - 承认数据局限（49-50%缺乏变异）
   - 承认测量局限（只看风险/薪资，看不到创新）

### ❌ 要避免的：
1. **不要简单说"多样性无用"**
   - 容易引起误解和争议

2. **不要忽视社会价值**
   - 多样性有公平、正义、社会责任等多重价值

3. **不要过度解读微弱相关**
   - r<0.03的相关性不足以得出确定结论
